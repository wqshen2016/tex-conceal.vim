# tex-conceal
A vim plugin for extended concealment of symbols in tex files.

## Installation 
```
Plug 'wqshen2016/tex-conceal', {'for': 'tex'}
```

## Settings
```
highlight clear Conceal
set conceallevel=2
let g:tex_conceal="abdgm"
```
